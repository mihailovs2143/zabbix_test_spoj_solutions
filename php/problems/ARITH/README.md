# ARITH - Simple Arithmetics

## Описание
Решение задачи SPOJ ARITH - арифметические операции над большими числами с форматированным выводом.

## Алгоритм
- **Сложение/Вычитание**: Посимвольная арифметика с переносом
- **Умножение**: Умножение на каждую цифру + сложение частичных произведений

## Особенности реализации
1. **Работа со строками** - числа до 500 цифр, int не подходит!
2. **Точное форматирование** - выравнивание по правому краю
3. **Частичные произведения** - каждое выравнивается по своей позиции

## Сложность
- **Время**: O(n*m) для умножения, где n,m - длины чисел
- **Память**: O(n+m) для хранения результата
